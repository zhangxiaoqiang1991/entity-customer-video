# merchant-video-script：本地商家短视频脚本生成器

输入商家信息，一键生成带拍摄建议的抖音口播脚本。覆盖餐饮/美容/汽修/教培四大行业，支持老板出镜和不出镜双模式。

## 核心信念

- **用流量做生意，不是做流量** — 找到客户，成交客户，方向对了油门才有效
- **20 秒定生死** — 前 3 秒找客户，后 17 秒成交客户
- **发什么就会来什么** — 你想要谁看到，开头就发什么
- **100 条之前不谈结果** — 损失有限，回报巨大，值得疯狂干
- **最好的动作是模仿** — 用已验证的结构，不用从零创新

## 核心架构：找到客户 → 成交客户

一条视频只做两件事：

| 段落 | 时长 | 做什么 |
|------|------|--------|
| **找到客户** | 1-3秒 | 地方+品类标签精准锁客 |
| **成交客户** | 4-20秒 | 产品理由（品质/服务）+ 人设理由（实在/靠谱） |

## 支持的内容类型

| 类型 | 目的 | 可批量？ | 难度 |
|------|------|---------|------|
| 标签型 | 锁定周边精准流量 | ✅ | 低 |
| 价值型 | 建立信任和人设 | ❌ | 中 |
| 避坑型 | 展示专业度 | ❌ | 中 |
| 对比型 | 建立差异化 | ⚡ | 中 |

## 支持的行业

- 餐饮（火锅、中餐、快餐、饮品……）
- 美容（美发、美甲、皮肤管理、医美……）
- 汽修（保养、维修、改装、洗美……）
- 教培（学科、艺术、编程、体育……）

## 两种模式

- **老板出镜**：第一人称脚本，老板面对镜头建立信任
- **不出镜**：纯 B-roll + 老板配音，适合不想出镜的老板

## 完整案例：成都德系车专修（不出镜）

```
行业：汽修 | 专修：奔驰/宝马/奥迪 | 开店：10年 | 出镜模式：不出镜
```

### Case 1：标签型

**结构：** 找到客户(地方+品类) → 成交客户(产品理由) → 成交客户(产品+人设理由) → 成交客户(行动号召)

| 时段 | 口播文案 | 画面 | 字幕 |
|------|---------|------|------|
| 1-3s | 在成都开了10年修车铺，没打过广告，全是车主一个介绍一个来的 | 门头→举升机上的奔驰宝马 | 成都·10年德系专修 |
| 4-7s | 我专修德系车，奔驰宝马奥迪，十年了我就干这一件事 | 三台品牌车在工位上 | 十年·只修德系 |
| 8-15s | 来我这的车主很多跟了我七八年。为啥不走？技术信得过，价格比4S店实在，不该换的绝不让你换 | 师傅检查零件/电脑检测 | 技术信得过·价格实在 |
| 16-20s | 成都开德系车的，左下角导航过来，到了报我名，送你一次全车检测 | 切门头 | 送全车检测↓ |

### Case 2：价值型

| 时段 | 口播文案 | 画面 | 字幕 |
|------|---------|------|------|
| 1-3s | 修了10年奔驰宝马，说几句同行不爱听的实话 | 焊接火花/发动机特写 | 10年修车·说几句实话 |
| 4-7s | 很多修车铺看到开豪车的进来，报价先翻一倍。我不干这种事 | 师傅在车底干活 | 看到豪车就加价？ |
| 8-15s | 该换的换，能修的不换，能给你省的钱我一定告诉你。10年了，我的客人全是这么来的 | 师傅检查零件放回去/老客户取车 | 能修不换·10年全是回头客 |
| 16-20s | 成都开德系车的，信得过我的，左下角导航过来。我在店里，来了直接开整 | 切门头 | 导航过来·直接开整↓ |

### Case 3：避坑型

| 时段 | 口播文案 | 画面 | 字幕 |
|------|---------|------|------|
| 1-3s | 开奔驰宝马的注意了，这3个保养项目真没必要去4S店 | 大字幕列出三个项目打叉 | 3个没必要去4S店的项目 |
| 4-7s | 第一个：发动机清洗，两三千，只要机油用得好根本不用洗。第二个：空调消毒，几百块管不了三天，换个好滤芯就行 | 拧机油盖/新旧滤芯对比 | 两三千→没必要/几百块→换滤芯 |
| 8-15s | 第三个：刹车油两年一换足够。这3项省下来一年少说省三五千。我是修了10年德系车的，专业不专业，你来看一次就知道 | 刹车油检测笔特写 | 刹车油·两年一换足够 |
| 16-20s | 左下角导航过来，免费帮你检查一遍，不收钱 | 切门头 | 免费检查·导航来↓ |

### Case 4：对比型

| 时段 | 口播文案 | 画面 | 字幕 |
|------|---------|------|------|
| 1-3s | 同样修宝马，4S店报8000，我收3000，差在哪？ | 两张工单对比：8000打叉/3000画圈 | 8000 vs 3000 |
| 4-7s | 4S店换总成，我修零件。一个密封圈的事儿，他们让你换整个发动机 | 新旧密封圈对比 | 换零件≠换总成 |
| 8-15s | 20年老师傅，只做德系。不是花不起8000，是不该花的钱一分都不多花 | 师傅检测/专注工作 | 20年德系专修·花对的钱 |
| 16-20s | 左下角导航过来，免费检测，先报价你再决定修不修 | 切门头 | 免费检测·先报价再修↓ |

## 快速开始

### 在 Claude Code 中使用

```bash
cp -r skills/merchant-video-script ~/.claude/skills/
# 然后在对话中输入：
/merchant-video-script
```

或在对话中直接说出：
- "帮我写个商家视频脚本"
- "我是开餐馆的，想做抖音"
- "实体店怎么拍短视频"

### 在 Workbody / Kimi Code / 任意 AI 工具中使用

直接把 `SKILL.md` 的内容粘贴为系统提示词，按提示填写商家信息即可。
