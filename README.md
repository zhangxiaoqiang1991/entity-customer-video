# entity-copywriter：实体获客短视频脚本生成器

输入商家信息，一键生成带拍摄建议的抖音短视频脚本。覆盖餐饮/美容/汽修/教培四大行业，老板出镜拍摄。

⚠️ **发布时务必打开位置定位，这样周边人群才能刷到。**

## 使用方式

```
安装 → 采访 → 生成
```

### 一行命令安装（以 Claude Code 为例）

```bash
cp -r skills/entity-copywriter ~/.claude/skills/ && /entity-copywriter
```

也支持以下 AI 编程工具（把 SKILL.md 内容粘贴为系统提示词即可）：国内：Workbuddy / Codebuddy / KimiCode / Trae　　国外：Codex / OpenClaw

### 三步使用

1. **安装** — 装好 SKILL.md
2. **采访** — 告诉我你的情况（在哪个城市、卖什么、有什么特点）
3. **生成** — 我出 20 秒脚本 + 拍摄建议，你用剪映或开拍 APP 拍了就发

## 底层逻辑

> **你做短视频不是为了做流量，是为了做生意的。**

一条 20 秒获客视频只做两件事：

| 段落 | 时长 | 做什么 |
|------|------|--------|
| **找到客户** | 1-3秒 | 地方+品类标签精准锁客 |
| **成交客户** | 4-20秒 | 产品理由（品质/服务）+ 人设理由（实在/靠谱） |

## 核心信念

- **用流量做生意，不是做流量** — 方向对了，油门才有效
- **20 秒定生死** — 前 3 秒找客户，后 17 秒成交客户
- **发什么就会来什么** — 你想要谁看到，开头就发什么
- **老板出镜** — 对着产品说，就这么简单
- **100 条之前不谈结果** — 损失有限，回报巨大，值得疯狂干

## 支持的内容类型

| 类型 | 目的 | 可批量？ | 难度 |
|------|------|---------|------|

## 案例展示（以餐饮行业为例）

以下展示同一家店（杭州西湖区重庆老火锅）的四种内容类型，让你直观看到不同选题的脚本差异。

### 案例 1：标签型（锁定周边流量·可批量）

**选题：在杭州，想吃正宗重庆火锅，来我家就对了**

| 时段 | 段落作用 | 口播文案 | 画面 | 拍摄建议 |
|------|---------|---------|------|---------|
| 1-3s | 找到客户 | 最近很多杭州的朋友刷到我的视频，刻意来我们家吃火锅 | 老板站店门口→店内环境 | 手机横屏拍门头，3秒内说完 |
| 4-7s | 成交客户 | 我们家开了3年，没打过广告，全是老客户一个带一个来的 | 店内客人吃火锅的热闹画面 | 高峰期拍，真实人气才有说服力 |
| 8-15s | 成交客户 | 用的鲜切牛肉，每天现熬底料，别的地方吃不到这个味。我做火锅就一个原则——实在 | 特写牛肉→沸腾锅底→老板切菜 | 手机人像模式拍牛肉纹理，暖色调 |
| 16-20s | 成交客户 | 想吃正宗重庆火锅的，左下角定位来就行，到了报我名，送你一份招牌菜 | 切门头，手指左下角 | 声音坚定，让观众觉得老板实在 |

### 案例 2：价值型（建立信任·适合老板出镜）

**选题：开了3年火锅店，说几句同行不爱听的实话**

| 时段 | 段落作用 | 口播文案 | 画面 | 拍摄建议 |
|------|---------|---------|------|---------|
| 1-3s | 找到客户 | 开了3年火锅店，说几句同行不爱听的实话 | 老板面对镜头，背景厨房热火朝天 | 开头用有冲击力的工作画面 |
| 4-7s | 成交客户 | 很多人做火锅底料用现成的、牛肉用冷冻的，我干不了这种事 | 现成料包→自己熬底料对比 | 对比画面最有说服力 |
| 8-15s | 成交客户 | 我们家牛肉早上6点送来，底料当天现熬。为什么80%是回头客？因为实在 | 牛肉特写→沸腾锅底→客人表情 | 牛肉要拍出"新鲜感"，油亮红色 |
| 16-20s | 成交客户 | 我在杭州西湖区，想吃口实在的，左下角定位过来 | 切门头 | 语速慢一点，稳一点 |

### 案例 3：避坑型（展示专业度·建立权威）

**选题：杭州吃火锅，这3种店千万别去**

| 时段 | 段落作用 | 口播文案 | 画面 | 拍摄建议 |
|------|---------|---------|------|---------|
| 1-3s | 找到客户 | 杭州吃火锅，这3种店千万别去 | 沸腾锅底画面，叠字幕列出三种打叉 | 制造"你被坑过吗"的好奇 |
| 4-7s | 成交客户 | 第一种：锅底免费的。锅底是灵魂，免费的东西能给你用好料吗？ | 现成料包 vs 自己熬底料 | 不点名品牌 |
| 8-15s | 成交客户 | 我们家锅底每天现熬，用什么料自己把关，3年没变过 | 熬底料→放辣椒花椒特写 | 拍放料、搅拌、看火候的过程感 |
| 16-20s | 成交客户 | 左下角定位过来，我请你试一口锅底，好不好吃你说了算 | 切门头，字幕"来试锅底" | CTA：先试锅底再点菜 |

### 案例 4：对比型（建立差异化·竞争激烈市场）

**选题：人均80的火锅和人均200的，到底差在哪？**

| 时段 | 段落作用 | 口播文案 | 画面 | 拍摄建议 |
|------|---------|---------|------|---------|
| 1-3s | 找到客户 | 杭州吃火锅，人均80和人均200的，到底差在哪？ | 左边高档装修，右边自己店热闹画面 | 左右对比就有冲击力 |
| 4-7s | 成交客户 | 差的不在食材，差在装修费和房租。200的一半在替房东付 | 远拍高档店门口→自己店菜单 | 不点具体店名 |
| 8-15s | 成交客户 | 我们家人均80，牛肉鲜切，底料现熬。朋友来了用好东西招待，收个实在价 | 牛肉特写→锅底→热闹就餐 | 轻松自信，不踩别人 |
| 16-20s | 成交客户 | 西湖区的，想吃实惠又正宗的，左下角定位过来，看看80块能吃到什么水平 | 切门头 | 用"看看80块能吃到什么"制造好奇心 |
